# data-iminsight
## Prerequisite
```
# node.js 설치가 되어있지 않은 경우 node.js 설치 (m1 노트북)
$ brew install node
# 잘 설치되었는지 node.js 버전 확인
$ node --version

$ git clone https://github.com/imwebme/data-iminsight.git

# 대시보드 실행
$ cd data-iminsight/iminsight-dashboard
$ npm install
$ npm run nodemon

# 서버 실행
$ cd data-iminsight/iminsight-server
$ npm install
$ node App.js
```
